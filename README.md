# Alif Nuryana's Dotfiles

This repository contains my personal configuration files for my Linux setup. These dotfiles help me to quickly set up a new system with my preferred settings and tools.

## Contents

- Shell configuration (using zsh with the Antigen plugin manager)
- Cross-shell prompt configuration (using Starship)
- Terminal emulator configuration (using Alacritty)
